db-swift

Requirements

  • Carthage
  • libmysqlclient

After that

Development

  1. Install mysql (which includes libmysqlclient) via brew install mysql
  2. Update carthage dependencies via
carthage update
  1. Create a branch based on git workflows
  2. Open xcode and start developing
  3. Commit your changes
  4. Squash your commits into a single commut and submit it via a PR
